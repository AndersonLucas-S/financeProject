<p>Projeto financeiro de gerenciamento de contas, criado em Laravel 5.5</p>
<p>Para rodar o projeto, é importante ter uma versão do php inferior a 8.</p>
<p>Após baixar o projeto em sua maquina certifique-se de que criou o arquivo .env e o banco de dados, após criar o banco de dados abra o projeto em um terminal
e então digite o comando: <b>php artisan migrate</b> para criar as tabelas no banco</p>
<p>Após criar as tabelas use o comando: <b> php artisan db:seed </b> para criar um usuário padrão de primeiro acesso.</p>
